## Contexto del proyecto

En este repositorio se encuentran algunas avances sobre nuestro proyecto integrador
un videojuego enfocado al transtorno opocionista desafiante, presente en niños de 
corta edad. La meta del proyecto es desarrollar un videojuego que apoye a la canalizacion
de los niños, y los acompañe a lo largo de su tratamiento.

## Clonando el repositorio

Mediante el powershell en windows, o la terminal correspondiente de tu sistema 
operativo, podemos clonar el repositorio de la siguiente manera:

``` bash
git clone https://github.com/ilyCereza/HerramientasdeGestion.git
```


##  Contenido del repositorio

La estructura del repositorio consta de 3 carpetas principales: 
    - Codigo: Dentro de ella, se encuentran referencias respecto a fragmentos de codigo fuente del proyecto
    - Documentacion: Se trata de documentos relacionados al desarrollo del proyecto
    - Recursos: Contiene diferentes archivos de imagen utilizados a lo largo del proyecto
